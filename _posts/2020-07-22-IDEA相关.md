## IDEA包编译出错

### 报错现象
```
Error:(5, 37) java: 程序包data.lab.elasticsearch.common不存在
```

### 处理方式一
在Terminal下执行如下操作，上述问题得以解决
```
mvn idea:idea
```
或者执行
```
mvn idea:module
```
### 处理方式二
```
Settings-->Build-->Build Tools-->Maven-->Runner-->勾选上Delegagte IDE build/run actions to Maven
```

### 打印GC日志
在VM处配置
```
-XX:+PrintGCDetails
```
