## 安装UV

```shell
pip install uv
```

## 同步依赖项到当前项目

```shell
uv sync
```

## 初始化项目，生成`.toml`文件

```shell
uv init
```

## 安装依赖并生成或更新`uv.lock`文件
```shell
uv pip install <package-name>@latest
uv pip install <package-name>==<version>
```

## 打包代码
```shell
# 会在dist目录下生成`.gz`和`.whl`文件
uv build
```

## 备注

1. project.toml文件与setup.py文件作用类似，都是python包打包发布时需要使用的文件
2. uv.lock文件的作用与requirement.txt类似
