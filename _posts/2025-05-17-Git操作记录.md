## 操作

```bash
# 撤销某次提交，会自动为你生成一个新的撤销提交
git revert <commit-id>
# 将撤销提交推送到远程
git push
```

```powershell
# 清除Windows凭据管理器中的凭据
# 查看所有凭据
cmdkey /list

# 删除与datalab.china.cn相关的所有凭据
cmdkey /delete "git:http://datalab.china.cn"
```
