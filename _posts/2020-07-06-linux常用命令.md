## 磁盘操作
### 查看磁盘剩余空间大小
```
df -h
```
### 查看某个目录占用的存储大小
```
du -ah --max-depth=1
```

### 查看文件大小
```
ll -lh import/hold-data.csv
```

### 查看文件行数
```
wc -l import/hold-data.csv
```

### 查看被占用的端口【3000端口】
```
netstat -tunlp |grep 3000
```
